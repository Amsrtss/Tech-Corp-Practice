# TechCorp SQL Practice Project

## Overview

This project contains SQL practice based on the TechCorp case study, covering data management for products, customers, orders, employees, and support tickets.

This work was completed as part of my learning journey from MySkill.id, focusing on:

* Basic to intermediate SQL queries
* Data aggregation
* Table joins
* Simple data analysis

---

## Database Structure

The database consists of the following main tables:

* **Products** → stores product data
* **Customers** → stores customer data
* **Orders** → stores transaction data
* **OrderDetails** → stores item-level order details
* **Employees** → stores employee data
* **SupportTickets** → stores customer support issues

---

## Features / What’s Inside

This repository includes:

### 1. Database Schema (DDL)

* Database and table creation
* Table relationships (foreign keys)

### 2. Sample Data (DML)

* Dummy data for testing
* Simulated business scenarios

### 3. SQL Queries

Example case studies solved in this project:

* Top customers based on total spending
* Average order value per customer
* Employees with the most resolved tickets
* Products that have never been ordered
* Total revenue calculation
* Product category analysis
